# 投注规则与回测假设

## 目标

阶段 7 的回测系统用于验证模型信号在历史赛果上的表现。它不是实盘下注系统，结果只用于研究和筛选策略。

## 支持的投注类型

- 独赢（`win`）：马匹最终名次为第 1 名时命中。
- 位置（`place`）：马匹最终名次为第 1、2、3 名时命中。

## 支持的注码策略

### 固定注码（`flat`）

- 每个候选投注使用固定注码。
- 默认每注 10。
- 若 bankroll 低于固定注码，则使用剩余 bankroll。

### 分数 Kelly（`fractional_kelly`）

- 使用公式：`kelly = ((odds - 1) * p - (1 - p)) / (odds - 1)`。
- 实际下注比例为 `kelly * kelly_fraction`。
- 默认 Kelly 分数为 0.25。
- 单注最大占 bankroll 比例默认限制为 2%。
- 若 Kelly 比例小于或等于 0，则跳过该投注。

## 候选筛选规则

- 按场次分组，每场最多选择 `top_n_per_race` 个候选。
- 独赢默认要求 `edge >= 0.03`。
- 位置投注如果缺少赛前位置市场概率，主要依赖模型位置概率和 `min_probability`。
- 排序优先级：
  - 独赢：先按 `edge`，再按模型胜率。
  - 位置：按模型位置概率。

## 结算规则

- 首选使用 HKJC 派彩表结算。
- 派彩表金额按每 10 港元派彩折算为 decimal odds，例如派彩 26.00 视为赔率 2.6。
- 若独赢缺少派彩表，回退使用赛果表中的 `win_odds`。
- 位置投注缺少 PLACE 派彩时跳过该投注。
- payout 包含本金，profit/loss = payout - stake。

## 指标定义

- ROI：总盈亏 / 总投注额。
- 命中率：命中投注数 / 投注数。
- 最大回撤：资金曲线相对历史高点的最大回撤比例。
- 盈亏因子：总盈利 / 总亏损绝对值；没有亏损时暂记为空。
- 平均赔率：所有已结算投注的平均 decimal odds。
- 投注次数：实际下注并完成结算的投注数。
- 资金曲线：按投注顺序记录每笔投注后的 bankroll。

## 当前假设

- 假设模型输出在赛前已经存在。
- 模型分两条线：`baseline-logistic-no_odds` 不使用任何赔率特征；`baseline-logistic-pre_start_odds` 使用旧赔率快照中“该场最终赔率快照时间往前 1 分钟”之前可见的最后一笔独赢赔率。
- 禁止默认使用赛果页最终独赢赔率训练模型；如需复现实验，只能显式使用 `odds_mode=result_final`。
- 假设投注都能按回测使用的下注时点赔率成交。
- 暂不计入交易成本、滑点、投注限额、赔率成交失败、盘口停牌和账户限制。
- 暂不处理同一组合下注之间的相关风险。
- 使用赛后派彩作为结算依据；禁止用赛后派彩或最终赔率反推投注时市场赔率。
- 回测结果在策略可信之前必须和这些假设一起展示。

## 推荐使用方式

1. 先用固定注码验证信号方向是否有效。
2. 再用分数 Kelly 检查资金曲线是否过度依赖少数高赔率样本。
3. 对每个模型版本分别回测，不混用不同模型版本。
4. 回测至少按赛日分段复核，避免单一赛日样本误导。
